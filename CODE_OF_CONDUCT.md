# Code of Conduct

## Our Pledge

VidyaSetu is committed to providing a welcoming, respectful, and inclusive environment for contributors.

Everyone participating in this project is expected to help create a space where people can learn, collaborate, ask questions, and contribute without fear of harassment or discrimination.

## Expected Behavior

- Be respectful and considerate in discussions.
- Assume good intent, especially when reviewing beginner contributions.
- Give constructive feedback that focuses on the work, not the person.
- Be patient with questions and setup issues.
- Respect different backgrounds, skill levels, identities, and experiences.
- Keep project spaces focused on collaboration and learning.

## Unacceptable Behavior

- Harassment, intimidation, insults, or personal attacks.
- Discriminatory language or behavior.
- Sexualized language, imagery, or attention.
- Trolling, deliberate disruption, or bad-faith arguments.
- Publicly sharing private conversations or personal information without consent.
- Any behavior that makes contributors feel unsafe or unwelcome.

## Reporting Issues

If you experience or witness unacceptable behavior, please report it to the project maintainer.

When reporting, include:

- What happened
- Where it happened
- Who was involved
- Any relevant screenshots, links, or context

Reports will be reviewed with care and handled as privately as reasonably possible.

## Enforcement

The maintainer may take action depending on the severity and context of the issue, including:

- A private warning
- Comment or issue removal
- Temporary restriction from project spaces
- Permanent removal from the project community

## Scope

This Code of Conduct applies to all VidyaSetu project spaces, including GitHub issues, pull requests, discussions, documentation, and contributor communication related to the project.

## Contributor Reminder

VidyaSetu is a learning-focused open-source project. Review feedback, questions, and discussions should help contributors improve while keeping the project healthy and respectful.
